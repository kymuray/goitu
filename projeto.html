Recursos Utilizados no Desenvolvimento do Site

Tipo de recurso Nome / Versão Link / Código Função / Uso no site
Fonte Google Fonts – Ruda https://fonts.googleapis.com/css2?family=Ruda:wght@400..900&display=swap Fonte principal do site, aplicada nos textos.
Ícones Bootstrap Icons v1.11.3 https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css Ícones usados nos cards (ex.: estrelas de avaliação).
CSS Bootstrap 4.1.3 https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/css/bootstrap.min.css Grid, responsividade, navbar, cards, botões.
JS jQuery 3.3.1 Slim https://code.jquery.com/jquery-3.3.1.slim.min.js Necessário para funcionalidades do Bootstrap (dropdown, navbar toggler).
JS Popper.js 1.14.3 https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.3/umd/popper.min.js Posicionamento de tooltips e dropdowns do Bootstrap.
JS Bootstrap 4.1.3 https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/js/bootstrap.min.js Funcionalidades do Bootstrap (navbar colapsável, modais, carrosséis).
